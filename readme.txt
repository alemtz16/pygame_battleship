# Proyecto de Juego de Battleship

## Descripción
Este es un juego de Battleship implementado con Pygame. Permite a los jugadores colocar sus barcos en una cuadrícula y jugar contra una IA que también posiciona sus barcos y hace movimientos óptimos.

## Instalación

### Requisitos Previos
- Python 3.x
- Pygame

### Paso a Paso

**Clonar el Repositorio**
   git clone https://github.com/analuciarojas/pygame_battleship.git
   cd pygame_battleship

*Crear un Entorno Virtual

python -m venv env
source env/bin/activate   # En Windows usa `env\Scripts\activate`

*Instalar dependencias

pip install pygame

*Compilador y IDE
Compilador: Python 3.x
IDE recomendado: Visual Studio Code (VS Code)

*Librerías Necesarias
El programa requiere las siguientes librerías para funcionar correctamente:
1. pygame


**Ejecución del Programa
Asegúrate de que tu entorno virtual esté activado.
Ejecuta el archivo principal del juego:

python main.py

*Notas Adicionales
Asegúrate de tener las imágenes y recursos necesarios en las rutas especificadas en el código.
Puedes personalizar la configuración del juego modificando los archivos de settings según tus necesidades.

¡Disfruta del juego!
